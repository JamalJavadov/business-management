# Business Management (Spring Boot)

Professional Spring Boot project for managing products, customers, and sales with a clean UI.

## Features
- Products with image URLs, purchase/sale prices, description, and type.
- Customers directory.
- Purchases tracked under each customer with active/completed status.
- Dashboard with monthly/yearly revenue statistics.
- Built-in H2 database and sample data loader.

## Run
```bash
mvn spring-boot:run
```

Open the app:
- http://localhost:8080

Optional H2 console:
- http://localhost:8080/h2-console

## Requirements
- Java 17+
- Maven 3.8+
